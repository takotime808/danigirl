Images for website. Convert images using photo manipulation utils:
```bash
python manipulate_images_for_web.py -m -c -oe webp -s "" -o "outdir" -i "2Q6A4289.jpeg" -sf 0.75
```
- `-m`: remove metadata
- `-c`: add copyright info
- `-oe`: output extension
- `-s`: output suffix
- `-o`: output directory
- `-i`: input filename
- `-sf`: scale factor
